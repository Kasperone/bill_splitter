# Bill Splitter

Share your bill with friends!

## Description

Bill Splitter app, simply splits the bill and tip between friends.
Made using Javascript during the Udemy course by Mateusz Maj.
* link to the course [Mateusz Maj Course](https://www.udemy.com/course/10-projektow-w-czystym-javascript-cz-1/) (the course is in Polish)

## Getting Started
### Dependencies

* You don't need any dependencies

### Installing

* Just fork, clone or download the project.

### Executing program

* Open index.html file in your browser.
